Madagascar Tourism Hub

A simple, infographic-first tourism hub for people who want to visit Madagascar. It includes high-level visa guidance and promotes **DMA tourisme** as a guided tourism operator.

## Local development

The web app lives in `web/`.

```bash
cd web
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Build

```bash
cd web
npm run build
npm run start
```
