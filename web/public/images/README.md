Drop brand assets and photos here.

Suggested structure (optional):

- `web/public/images/brand/` (logo, icons)
- `web/public/images/hero/` (hero photos)
- `web/public/images/sections/` (visa/tours/stays)

Once files are here, they can be used from code like:

- `/images/brand/logo.png`
- `/images/hero/antananarivo.jpg`
