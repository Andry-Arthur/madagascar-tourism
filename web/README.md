Madagascar Tourism Hub (web)

This is the Next.js app for the Madagascar Tourism Hub.

## Getting started

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
npm run start
```

## Notes

- The design direction is intentionally minimal: white background, black text, with colored SVG/PNG accents.
- Replace placeholder assets in `public/` with real photos/brand assets when ready.
